const GLint OmniOGL::OGLUniforms::diffuse_color
private

A shader program uniform location. This uniform holds a single float value which represents the amount of diffuse lighting that appears on the rendered object, as a percentage value. This value can range from 0.0 to 1.0, with 1.0 showing the most light.

Diffuse lighting is the bread-and-butter of lighting calculations and will result in a soft glow off of surfaces as if they were reflecting the surrounding lights accurately.

const GLint OmniOGL::OGLUniforms::specular_color
private

A shader program uniform location. This uniform holds a single float value which represents the amount of specular lighting that appears on the rendered object, as a percentage value. This value can range from 0.0 to 1.0, with 1.0 showing the most light.

Specular highlight is best described as a "reflective sheen". Note that specular highlights use the uniform uni_lshininess to determine the radius of the resulting reflective "flare".
