Section * sections
 A pointer to all Sections which partition the Scene's 3D space and allow chunks which are not in close proximity to the player to be hidden. In Scene::Draw(), certain elements in this array will be accessed via their index and their objects will rendered. When an object is added to the Scene, RenObj::origin and RenObj::en_internal_clipping are used to determine whether or not the object should belong to a Section, and if so, which Section the object is added to.

Section* OmniOGL::SectionCollection::sections
private

A pointer to all Sections which partition the Scene's 3D space and allow chunks which are not in close proximity to the player to be hidden. In Scene::Draw(), certain elements in this array will be accessed via their index and their objects will rendered. When an object is added to the Scene, RenObj::origin and RenObj::en_internal_clipping are used to determine whether or not the object should belong to a Section, and if so, which Section the object is added to.

See Also
Scene::section_count, Scene::AddObject()

Initialized to be an array of X number of Sections within the Scene constructor. This pointer is dynamically allocated at runtime and must be deleted in the Scene's destructor. All sections should be contiguous.
