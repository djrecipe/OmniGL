bool OmniOGL::MaterialLighting::black_only

Used to force an object to all black. Currently this is used to force text to all black should the font size be too small, since in the current implementation of rendering text there is a white border around black text.

Initialized to False by default.

GLfloat OmniOGL::MaterialLighting::diffuse_factor[4]

Four floats which represent the r,g,b and alpha diffuse factors. Each factor should be a number between 0.0 and 1.0, as they are essentially a percentage that imply "this percentage of the r,g,b, or a component of diffuse lighting that 'hits' this object will be visible".

See Also
RenObj::factor_vec4()

Each element is initialized to 1.0 by default, which means the object will show 100% of the diffuse lighting "cast upon" it. Diffuse lighting generally looks soft and gradual.

GLfloat OmniOGL::MaterialLighting::shininess

A single float which ultimately ends up being an exponent when calculating specular lighting in the fragment shader. Increase this number to make the specular reflection tighter, brighter, and more concise - almost as if it were perfectly reflecting a spherical light source. Decrease the number to broaden the effect.

Initialized to 100 in the MaterialLighting default constructor. (100 was an arbitrary pick.)

GLfloat OmniOGL::MaterialLighting::specular_factor[4]

Four floats which represent the r,g,b and alpha specular factors. Each factor should be a number between 0.0 and 1.0, as they are essentially a percentage that imply "this percentage of the r,g,b, or a component of specular lighting that 'hits' this object will be visible". Specular lighting is synonmous with "Phong Shading" in this rendering engine.

See Also
RenObj::factor_vec4()

Each member is initialized to 1.0 by default, which means the object will show 100% of the specular lighting "cast upon" it. Specular lighting generally looks shiny and reflective.
