namespace  OmniOGL
 The primary namespace which encompases all the functions and variables necessary to render and manipulate a scene using OpenGL. This namespace spans several files and deals not only with the issue of rendering, but also the issue of user input, environment generation and interaction, and [eventually] physics.

GLuint OmniOGL::MakShadProg (GLuint vertexShader, GLuint fragmentShader)
 Attempts to construct a shader program using an already-compiled Vertex Shader and Fragment Shader. This method attaches the Vertex and Fragment Shaders via glAttachShader() then "activates" them via glLinkProgram() and glUseProgram(). Finally this method "connects" the Vertex Shader to the Fragment Shader by assigning an output variable from the Vertex Shader to an input variable in the Fragment Shader.
