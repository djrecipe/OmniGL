OmniOGL::NetNode Class Reference

A single node in a collection of nodes (SurfaceNet). There is one node for each vertex in the mesh and "duplicate" vertices will be considered the same node. This node holds data such as its coordinates, pointers to neighbors (any other NetNodes with whom this NetNode shared a triangle with), the element index of said neighbor, and the node's constraint value. Currently spherical constraints are implemented which means the constraint value will be equal to half the distance to [this] NetNode's closest neighboring NetNode.

bool OmniOGL::NetNode::Relax ( void  )

Iterates through all neighbors of this NetNode and determines the average distance this NetNode lies from all of its neighbors (vertices with whom this NetNode shared a triangle). A NetNode cannot be moved further than 90% of its constraint value (half the distance to the closest neighbor), thus two NetNode's should never be able to "collide". Also note that the resulting "relaxed" vertex is stored in NetNode::new_vertex so-as not to affect the relaxation process. Once all NetNodes are relaxed, then they can all be iterated through again so that calls to NetNode::CommitNewVertex() and then NetNode::UpdateConstraints() can be made.

See Also
NetNode::vertex, NetNode::new_vertex, NetNode::CommitNewVertex(), NetNode::UpdateConstraints()

Returns
A boolean value indicating whether or not the NetNode was moved at all. Will always return True unless the distance between this NetNode and its closest neighbor is twice the value represented by NETNODE_PROXIMITY_LIMIT.

This method should be called on all NetNodes in a given SurfaceNet before any all calls to NetNode::CommitNewVertex() or NetNode::UpdateConstraints() are made.
